xgettext 
------------------
Extracts i18n strings from sources inside _() macro
like printf( _("This is i18n string") );
Usage: xgettext -i <input file> -k=_ -o <output file>

msginit
------------------
Create .po initial file from template, which has been created by Deploy.js & xgettext
use it to start a new language localization from simple <input file>
Usage: msginit -i <input file> -o <output file>

msgmerge
------------------
Merge two .po files? <update-file> usually manually translated file, 
<ref-file> is generated by xgettext or extracted from .xrc
Usage: msgmerge.exe <update-file> <ref-file> -U

msgcat
------------------
Concatenate two or more .po files
Usage: msgcat <file1> .. <fileN> -o <file>

msgfmt
------------------
Compile .po file to machine dependent .mo file
Usage: msgfmt <file1> .. <fileN> -o <file>

Main usage strategy
1)xgettext the newest string resource file
2)msgmerge with user the old translation file for each locale
3)translate
4)msgftm
5)distribute
