%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\section{Приложение 1. Список параметров алгоритма}
\label{sec:appendix1}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\subsection{Параметры, изменяемые пользователем}
\begin{enumerate}
\item Чувствительность к движению. Число от 0 до 1, 0 -- низкая
чувствительность, 1 -- высокая чувствительность.
\end{enumerate}

\subsection{Параметры, недоступные для пользователя (доступны для разработчиков)}

Примечание: по английским комментариям легко обнаружить переменные с
этими параметрами в программе.

\begin{enumerate}

\item Скорость обновления фона (background update rate). Число от 0.001 до
1; значение по умолчанию 0.05.

\item
Время существования дырки (timeout for a hole existence in seconds).
Число от 1 до 100; значение по умолчанию 3.

\item
Уровень сигнал/шум (signal/noise threshold ration for change
detection). Число от 1 до 100; значение по умолчанию 3. Этот
параметр -- константа $С$ из пункта 4 раздела
\ref{sec:motiondetect}.

\item Минимальный периметр региона движения(the minimal perimeter of detected motion
region). Число от 1 до 1000; значение по умолчанию 10.

\item Минимальная площадь региона движения (the minimal area of detected motion region).
Число от 1 до 10000; значение по умолчанию 64.

\item Максимальное количество рамок на одном кадре(maximal number of rectangles on one frame).
Число от 1 до 16; значение по умолчанию 3.

\item Максимальное время прерывания события движения, в секундах (maximal time to motion pause for one event, in
seconds). Число от 0.1 до 5, значение по умолчанию 1.

\item Максимальное время жизни события движения, в секундах (maximal time to event to live, in seconds).
Число от 0.1 до 60, значение по умолчанию 7.

\end{enumerate}
